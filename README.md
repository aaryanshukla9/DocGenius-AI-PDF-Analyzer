# DocGenius: AI-Powered PDF Analyzer

DocGenius is an intelligent document analytics platform that transforms static PDF data into interactive, actionable insights. By leveraging **Retrieval-Augmented Generation (RAG)** and Large Language Models (LLMs), the application enables context-aware semantic search, instant Q&A, and deep information extraction through a clean, user-friendly interface. 

Instead of manually scrolling through dense pages or relying on basic keyword matching, users can upload complex documents and instantly query them using natural conversational language.

## 🚀 Key Features
* **Semantic Document Search:** Understands user intent and document context deeply, moving beyond simple keyword matching.
* **Deterministic Source-Backed Answers:** Minimizes AI hallucinations by forcing the model to strictly ground its responses in the extracted text of the uploaded PDF.
* **Intelligent Data Extraction:** Rapidly extracts core metrics, summaries, and complex insights from multi-page reports.
* **Streamlit UI/UX:** A responsive web application dashboard built for efficient human-AI interaction.

## 🛠️ Technical Stack
* **Frontend/Interface:** Streamlit (Python-native framework)
* **Core Language:** Python 3.x
* **AI Engine:** OpenAI API (GPT models & Semantic Embeddings)
* **Architecture:** Retrieval-Augmented Generation (RAG) pipeline
